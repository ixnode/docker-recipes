# Useful docker commands

## Start app

```bash
❯ docker-compose up -d
```

## Show logs

```bash
❯ docker-compose logs -f
```

## Show container

```bash
❯ docker container ls
```

## Login into container

```bash
❯ docker-compose exec nginx bash
```

## Restart, stop and start container:

```bash
❯ docker-compose restart
❯ docker-compose stop
❯ docker-compose start
```

## Shutdown container:

```bash
❯ docker-compose down
```
